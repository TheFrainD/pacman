## Getting Started

### Prerequisites

- C Compiler
- CMake
- OpenGL

1. Clone the repository:
   ```bash
   git clone https://github.com/TheFrainD/pacman.git
   cd pacman
   ```

2. Create a build directory and navigate into it:
   ```bash
   mkdir build
   cd build
   ```

3. Configure the project using CMake:
   ```bash
   cmake ..
   ```

4. Build the project:
    ```bash
    make
    ```

5. Run the executable:  
    ```bash
    ./pacman
    ```
